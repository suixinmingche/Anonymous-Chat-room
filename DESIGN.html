一项目概述

1.1 项目简介

匿名聊天室是一款专注于隐私保护的临时对话工具，包含两大核心功能模块：

💬 匿名聊天室 — 随机房间号 + WebSocket 实时聊天
🍶 飘飘瓶 — 匿名留言 + 口令保护 + 阅后即焚

二核心特性

🚫 无注册 — 无需账号、手机号、邮箱
🗑️ 无记录 — 不存储聊天记录
🎭 完全匿名 — 随机生成昵称
🔥 阅后即焚 — 消息自动销毁
🔐 口令保护 — 飘飘瓶密码加密
⏰ 24h 过期 — 飘飘瓶自动销毁
📷 图片发送 — 支持 Base64 图片
📱 响应式 — 适配桌面端与移动端
🌙 暗色/亮色 — 支持系统主题切换

六匿名聊天室

6.1 进入房间

🎯 入口方式

  • 随机进入：点击「随机进入房间」，前端生成 6 位随机房间号
  • 指定进入：输入 4-8 位房间号，点击「进入」

房间号规则

属性值
长度4-8 位
字符仅允许 a-z 和 0-9（小写化处理）
格式/^[a-z0-9]{4,8}$/

昵称生成

格式：[形容词] + [名词]（如「沉默的狐狸」「流浪的繁星」）

  • 形容词库：30 个（如「温柔的」「孤独的」「遥远的」）
  • 名词库：30 个（如「狐狸」「灰狼」「白鲸」）
  • 重复检测：与当前房间已用昵称去重

十API 文档

10.1 创建留言

POST /api/messages
Content-Type: application/json

Request Body:
{
  "content": string,           // 留言内容（必填，≤5000字）
  "self_destruct": boolean,    // 是否阅后即焚
  "destruction_delay": number, // 销毁延迟秒数（默认10）
  "password": string | null   // 口令（可选，≥4位）
}

Response 200:
{
  "id": string,               // 留言ID（8位）
  "expires_at": string,       // 过期时间（ISO格式）
  "has_password": boolean     // 是否有口令保护
}

Response 400: { "error": "留言内容不能为空" }
Response 429: { "error": "操作太频繁，请稍后再试" }

10.2 读取留言

GET /api/messages/:id
GET /api/messages/:id/:password

Response 200:
{
  "id": string,
  "content": string,
  "created_at": string,
  "expires_at": string,
  "self_destruct": boolean,
  "first_viewed_at": string | null,
  "replies": Array<{ "id", "content", "created_at" }>
}

Response 401: { "error": "需要口令", "need_password": true }
Response 404: { "error": "留言不存在或已失效" }
Response 410: { "error": "该留言已销毁", "destroyed": true }

10.3 验证口令

POST /api/messages/:id/verify
Content-Type: application/json

Request Body: { "password": string }

Response 200: { "success": true, "verified": true }
Response 401: { "success": false, "error": "口令错误" }

10.4 发送回复

POST /api/messages/:id/replies
Content-Type: application/json

Request Body:
{
  "content": string,  // 回复内容（≤2000字）
  "password": string  // 若主留言有口令则必填
}

Response 200:
{
  "success": true,
  "reply": { "id", "content", "created_at" }
}

10.5 销毁留言

DELETE /api/messages/:id

Response 200: { "success": true, "message": "留言已销毁" }
Response 404: { "error": "留言不存在" }
